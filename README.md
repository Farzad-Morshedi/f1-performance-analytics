# F1 Performance Analytics: Predictability vs. Grit

## 🎯 Project Discovery & Objectives
This repository contains a two-part analytical study of Formula 1 racing dynamics using the Ergast Motor Racing Database.

### Project A: The Grid-to-Podium Predictor
**Objective:** Quantify the influence of qualifying rank on final race results across different circuit profiles.
*   **Method:** Linear Regression.
*   **Key Question:** How does the correlation ($R^2$) change between a street circuit (Monaco) and a high-speed circuit (Monza)?

### Project B: The Midfield King Metric
**Objective:** Identify the most efficient overtakers by calculating net positions gained, adjusted for mechanical reliability.
*   **Method:** Feature Engineering & Data Aggregation.
*   **Key Question:** Which drivers consistently outperform their car's qualifying pace when DNFs are removed from the equation?
